# Header-only-skeleton
[![Build Status](https://travis-ci.org/As-12/Header-only-skeleton.svg?branch=master)](https://travis-ci.org/As-12/Header-only-skeleton) [![Coverage Status](https://coveralls.io/repos/github/As-12/Header-only-skeleton/badge.svg?branch=master)](https://coveralls.io/github/As-12/Header-only-skeleton?branch=master) [![Build status](https://ci.appveyor.com/api/projects/status/awda4lv9md6wv8hg?svg=true)](https://ci.appveyor.com/project/As-12/header-only-skeleton) [![Documentation](https://codedocs.xyz/As-12/Header-only-skeleton.svg)](https://codedocs.xyz/As-12/Header-only-skeleton/)

 A skeleton framework for Header-only C++ library with Unit-testing and Benchmarking framework.
 This project integrates with Travis-CI to compile and run-unit test on gcc and clang.
 AppVeyor is used to compile a demo on Visual Studio 2017.

# Feature

- Unit Testing Framework is GoogleTest.
- Performance Testing Framework is GoogleBenchmark.
- Generate Documentation through Doxygen.
